Sobre o Investsimples:

O Investsimples nasceu a partir da necessidade da criação de uma ferramenta simples que pudesse demonstrar, de forma prática, como funciona o mercado de ações brasileiro. Esta ferramenta consiste na compra e venda de ações, utilizando os valores reais, para que seja possível visualizar as variações de preço e, assim, simular estas operações.
